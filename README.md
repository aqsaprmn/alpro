Installation

System Requirements
----
Most current web servers with PHP & MySQL installed will be capable of running API PINTER

Minimum Requirements
- Linux or Windows OS
- PHP Version 5.3+
- Both PDO & MySQLi Support
- GD2 Image Library
- MySQL Version 4.1.x and above

can be Installed in Raspberry Pi Device.
